# 题目
给定一个数组arr 代表每个人的能力值 再给定一个非负数k
如果两个人到能力差值正好为k 则可以一起比赛
一局比赛只有两个人
返回最多可以同时有多少场比赛

# 思路

## 贪心

### 代码实现
```java
private static int process(int[] array, int k) {  
   if (array == null || array.length < 2 || k < 0) {  
      return 0;  
   }  
   // 首先对数组进行排序
   Arrays.sort(array);  
   int res = 0;  
   int length = array.length;  
   // 左右指针
   int left = 0, right = 0;  
   // 存储已经参与比赛的下标
   boolean[] usedIndex = new boolean[length];  
   while (left < length && right < length) {  
	  // 如果当前位置已经参加比赛,则跳过
	  // 由于右侧指针在前,所有只需要对左侧指针进行判断 
      if (usedIndex[left]) {  
         left++;  
      } else if (left == right) {  
	    // 当左右指针相同时,右指针向前移动
         right++;  
      } else {  
	    // 求两个指针直接的差值
         int t = array[right] - array[left];  
         // 如果与k相等,则可以一起比赛,数量加一,并进行存储
         if (t == k) {  
            res++;  
            usedIndex[right] = true;  
            right++;  
         } else if (t < k) {  
            // 由于数组有序,且两者差值小与k,则右指针移动
            right++;  
         }else {  
	        // 大于k时,左指针移动
            left++;  
         }  
      }  
   }  
   return res;  
}

```
