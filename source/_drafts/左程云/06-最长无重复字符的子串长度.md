# 题目
给定一个字符串, 求其中最长无重复字符的子串长度

# 思路

## 动态规划

### 代码实现
```java
private static int maxLength(String s) {  
   char[] chars = s.toCharArray();  
   // 存储字符上一次出现的位置  
   int[] charIndex = new int[256];  
   Arrays.fill(charIndex, -1);  
   // 上一个位置无重复字符串的起始位置  
   int preStart = -1;  
   int res = 0;  
   for (int i = 0; i < chars.length; i++) {  
      // 当前位置的起始位置  
      // 由两个元素决定：      
      // 1. 当前元素上一次出现的位置      
      // 2. 上一个位置的开始位置      
      // 取两者的最大值作为此次的起始位置      
      int curStart = Math.max(charIndex[chars[i]], preStart);  
      int curLength = i - curStart;  
      res = Math.max(res, curLength);  
      preStart = curStart;  
      charIndex[chars[i]] = i;  
   }  
   return res;  
}

```
