# 题目
现有司机n\*2人 现在调度中心会把司机平均分配到A，B两个区域
第i个司机去A可获得收入income\[i]\[0]
第i个司机去A可获得收入income\[i]\[1]
返回所有调度方案中 能使所有司机总收入最高的方案，是多少钱

# 思路

## 动态规划
### 代码实现
```java
private static int maxMoney(int[][] income) {  
   if (income == null || income.length < 2 || income.length % 2 == 1) {  
      return 0;  
   }  
   int total = income.length;  
   int half = total >> 1;  
   return process(income, 0, half);  
}  
  
/**  
 * 从index位置开始，后续的司机向A，B区域分配 * A区域还能分配restCountOfA个人 * 返回所有收入最大值 * * @param income  
 * @param index  
 * @param restCountOfA  
 * @return  
 */  
private static int process(int[][] income, int index, int restCountOfA) {  
   // 所有司机分配完成  
   if (index == income.length) {  
      return 0;  
   }  
   // 还剩下司机未分配  
   // 当剩下的人数与还需要分配到A的人数一致时，剩下所有的人都分配到A中   
   if (income.length - index == restCountOfA) {  
      return income[index][0] + process(income, index + 1, restCountOfA - 1);  
   }  
   // 当还需分配到A区域的人数为0时，剩下所有人都分配到B区域  
   if (restCountOfA == 0) {  
      return income[index][1] + process(income, index + 1, restCountOfA);  
   }  
   // 当前司机可以去A，也可以去B，取其最大值  
   int caseA = process(income, index + 1, restCountOfA) + income[index][1];  
   int caseB = process(income, index + 1, restCountOfA - 1) + income[index][0];  
   return Math.max(caseA, caseB);  
}
```
